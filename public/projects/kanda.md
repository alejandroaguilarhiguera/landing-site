# Kanda

Desarrollo de App · Combate el acoso telefónico y telemarketing no deseado.

## Desafío

Este proyecto nace de la visión de utilizar tecnologías avanzadas para crear una comunidad segura y protegida de extorsionadores, fraudes y telemarketing telefónico no deseado.

## Logros

Siguiendo las especificaciones iniciales, desarrollamos un conjunto de aplicaciones para **iOS y Android**, aprovechando las funciones nativas de ambos sistemas. Creamos un directorio robusto con una lista negra y una lista blanca para las llamadas, permitiendo a los usuarios decidir con quién desean comunicarse.

Además, desarrollamos una plataforma administrativa que facilita a los gestores controlar los números reportados para revisión manual y mantener un registro detallado del crecimiento de usuarios y diversas métricas. Todo esto está interconectado a través de una API nuclear, que maneja la comunicación entre las aplicaciones, almacena y consulta datos, y notifica a los usuarios cuando es necesario.

## Tecnologías implementadas

- **API**: Utilizamos Node.js, Sequelize y Express para construir una API robusta y eficiente que maneja la comunicación entre las aplicaciones, gestiona el almacenamiento y consulta de datos, y envía notificaciones a los usuarios cuando es necesario.
- **BackOffice**: Desarrollamos una plataforma administrativa utilizando Next.js y Redux, permitiendo a los administradores controlar los números reportados, revisar manualmente las incidencias y monitorizar el crecimiento y las métricas de los usuarios.
- **App para iOS**: Implementamos la aplicación para iOS utilizando React Native y Swift, integrando módulos nativos que garantizan un rendimiento óptimo y una experiencia de usuario fluida.
- **App para Android**: Para la aplicación de Android, utilizamos React Native y Java, asegurando la integración de módulos nativos y un rendimiento eficiente en dispositivos Android.
